Tags HTML mais comuns

Elemento Semântico:

Elemento semântico é aquele que mostra em seu nome o que deve fazer e suas características

De bloco:

Elementos que não aceitam em sua essência outros elementos dos seus lados e preenchem um bloco como um todo

  • h1- h6
    • Seis Variações de escrita de cabeçalho. O h1 possui o maior tamanho de fonte, enquanto o h6 o menor.
  • div
    • O conteúdo de uma página é geralmente dividido em blocos, especificados pela tag div.
  • form
  • p
    • Texto sem formatação é colocado dentro desta tag, ou no caso paragrafo
  • ul/ol

De linha:

Elementos que aceitam em sua essência outros elementos dos seus lados e nao preenchem um bloco como um todo

  • img
  • span
  • a
  • button
  • input

Aprendendo Forms

  • Forms -> Utilizado principalmente para coletar dados dos usuários.
  • Input(text) -> Receber texto do usuário, campo de texto aberto
  • Radio - Para escolher uma dentre as opções
  • Checkbox - Para escolher mais de uma dentre as opções
  • Submit - Botão para enviar os dados preenchidos
